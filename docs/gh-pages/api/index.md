# API Documentation

This section documents all the ways you can interact with the Relational Wallet API, including what information you need to provide and what responses you'll receive.

- **Web Enclave API** — Attested RPC endpoints exposed from the enclave.
- **Wallet Web API/SDK** — Client-facing REST endpoints, events, and SDK helpers consumed by the UI.
