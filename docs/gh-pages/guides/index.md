# User Guides
{: .fs-8 }

Step-by-step workflows for the most common Relational Wallet operations.
{: .fs-5 .fw-300 }

---

## Available Guides

### [Wallet Management](/relational-wallet/guides/wallet-management)
Create your first wallet, label and organise wallets, and manage your address book.

### [Sending & Receiving](/relational-wallet/guides/sending-receiving)
Send AVAX and rEUR to addresses or email recipients, generate payment links, and read your transaction history.

### [Fiat On/Off-Ramp](/relational-wallet/guides/fiat-integration)
Deposit euros via bank payment to receive rEUR, and withdraw by burning rEUR to a bank account.

---

## Before You Begin

All operations require:

1. **A Clerk account** — sign up at `http://localhost:3000/sign-up`
2. **A running backend** — see [Installation](/relational-wallet/installation)
3. **A funded wallet** — use the [Fuji Faucet](https://faucet.avax.network/) for testnet AVAX

Browser users access everything through the web UI at `http://localhost:3000`. Developers can call the API directly — see the [API Reference](/relational-wallet/api).
